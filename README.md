## About

CS undergraduate based in Karachi, Pakistan. I build full-stack web applications with a focus on clean architecture, secure authentication, and real-world usability. My primary stack is MERN — React on the front, Node/Express on the back, MongoDB for data.

On the side, I work in cybersecurity: network scanning, web app pentesting, and CLI tooling. Four active industry certifications across both disciplines. I write code between midnight and fajr, drink too much coffee, and believe software should be both hard to break and easy to maintain.

Currently building toward production-grade applications and open to full-stack roles, collaborations, and freelance work.

### Commerce Nexus
`React` `Node.js` `Express` `MongoDB` `JWT`

Full-stack e-commerce platform with a complete order pipeline. JWT-based authentication, product and cart management, and a RESTful API backend. Built with a clean separation between client and server.

### Ghost Scanner
`Python` `Nmap` `Bash`

CLI recon tool for host discovery, port scanning, and vulnerability enumeration. Automates the early stages of network reconnaissance and outputs structured HTML reports. Built for use in controlled lab environments.

### Signal — Chat Protocol
`Node.js` `Socket.io` `Express`

Real-time multi-room messaging application built on WebSockets. Supports live user presence, room management, and a clean minimalist UI. Handles concurrent connections across multiple rooms simultaneously.
